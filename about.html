About

The name is derived from synonoms of 'Fizz' and 'Buzz'. In this case, 'carbonated' was used in place of 'Fizz' and 'rattle' was used in place of 'Buzz'

The background is courtesy of this codepen: https://codepen.io/Momika/pen/ydaWmQ

This website uses Prism.js to display the Javascript and CSS file contents.

This website uses Javascript to implement the FizzBuzz algorithm and to update some of the display elements.

Bootstrap 5 is used to give responsiveness to the website.

Certain elements of the website are styles with CSS

HTML 5 was used in providing the structure of this website.

Microsoft Visual Studio Code was the IDE used to create this entire project.
